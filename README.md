# DIKWP HumanReinvention Studio OS

**A practical operating system for proactive human reinvention in the age of AI.**

中文定位：**DIKWP 人类主动革新与 AI 共进化工作台**。

## Core proposition

The system does not ask people to compete with AI on speed, memory, or output volume. It helps a person, team, educator, healthcare worker, public-service manager, or founder redesign work around what becomes more valuable as AI becomes more capable:

- problem framing;
- evidence judgement;
- accountable decisions;
- trust and relationships;
- embodied and real-world action;
- cross-domain synthesis;
- public-value creation;
- proof, portability, and option resilience.

The renewal loop is:

```text
Sense -> Unlearn -> Reframe -> Co-design -> Experiment -> Prove -> Compound
```

## Quick start

### Standalone studio

Open `index.html` directly. It requires no account, API key, network, database, or server.

### Python CLI

```bash
pip install -e .
humanreinvention analyze examples/sample_reinvention_case.json --out outputs/demo
humanreinvention guard "build a three-role transition portfolio"
humanreinvention static-audit src --out outputs/demo/static_boundary_audit_report.json
```

## Governance boundary

This system is for voluntary learning, mentoring, organisational development, and reversible transition experiments. It is not a hiring screen, admissions system, psychological diagnostic, credit or insurance score, medical device, or employment guarantee.

It must not be used for:

- automatic hiring, firing, promotion, compensation, admissions, or discipline;
- covert monitoring, emotional manipulation, or punitive ranking;
- fabricated credentials or hidden AI use in evaluated work;
- medical, legal, financial, or crisis decisions without qualified human review;
- credential capture, market manipulation, or platform bypass.

## Strategic open-source model

The open core includes the schema, scoring logic, CLI, browser studio, synthetic scenarios, tests, and governance boundary. Potential official value layers include local calibration, mentoring programmes, institutional pilots, signed DIKWP HumanReinvention reviews, facilitator training, and TrustPassport registration.
